# Q4 核心成果：最佳赛制机制设计 (Mechanism Design)

## 1. 核心任务
设计一套“最优”的比赛规则（权重分配 + 淘汰机制），在保证**技术公平性**（不淘汰高分选手）的同时，最大化**商业留存率**（不淘汰高人气选手）。

## 2. 优化方法
*   **多目标网格搜索 (Multi-Objective Grid Search)**：
    *   升级为 **Step=0.05** 的精细化全域搜索。
    *   构建了包含“误杀率”、“留存率”、“悬念度”的综合损失函数。

## 3. 颠覆性发现：粉丝中心论 (Fan-Centric Optimality)
我们的模型推翻了传统的“50/50平衡论”。数据证明，为了节目整体利益，**必须赋予粉丝更大的话语权**。

### 最优动态权重方案 (The "Fan-First" Strategy)
*   **阶段一：前中期 (Week 1-8)**
    *   **权重**：**10% 评委 / 90% 粉丝**
    *   **逻辑**：在比赛初期，选手的去留应几乎完全取决于观众喜爱度。评委的过度干预极易导致有人格魅力的“收视率担当”过早出局。此时技术瑕疵是可以被容忍的。
*   **阶段二：决赛圈 (Week 9+)**
    *   **权重**：**45% 评委 / 55% 粉丝**
    *   **逻辑**：进入决赛后，权重回归平衡。此时必须提升评委话语权，以确保最终冠军具备令人信服的专业舞蹈水准，避免“水货冠军”争议。

### 兜底机制 (The Safety Net)
*   **Bottom 3 Judge Save**：
    *   在任何阶段，若技术分排名前 3 的选手跌入淘汰区，**评委拥有强制复活权**。
    *   该机制将“技术高分误杀率”从 5% 降低到了 **<0.5%**，完美弥补了高粉丝权重可能带来的技术不公。

## 4. 结论
最佳赛制不是简单的“公平”，而是**“前期顺应民意，后期尊重专业”**的动态平衡。这套方案能最大程度兼顾收视率与比赛的专业权威性。
